Import ZoomCharts custom visuals from Power BI marketplace

  1. Open Power BI services and login with your credentials, open an existing report or create a new one, where you would like to add a custom visual.

    OR

    Open an existing .pbix file (Power BI report file extension), login if necessary or create a new report with Power BI Desktop

  2. Click on the the ellipses at the bottom of the Visualizations pane.

  3. From the dropdown, select  Import from marketplace .

  4. Enter “ZoomCharts” in the search menu and press Enter.

  5. To learn more about the visual of your choice, hover mouse on the description and click on it.

  6. On the information page you can view screenshots, videos, detailed description and more.

  7. Scroll to the bottom to see user reviews.

  8. Import custom visual by clicking on Add within the search results list.

    or in the custom visual information page.

  9. The icon for the custom visual is added to the bottom of your Visualizations pane and the visual is now available for use in your report.

Download and import custom visuals from Microsoft AppSource

  1. Open  Microsoft AppSource in your browser. AppSource displays a tile for each custom visual. Each tile has a snapshot of custom visual, provides a brief description and a download link. To see more details, select the tile.

  2. On the detail page you can view screenshots, videos, detailed description and more. Download the custom visual by selecting “ Get it now”.  To download the visual successfully, you will need to agree to the Terms of Use.

  3. Click on the button to download the custom visual.

  4. Import the '.pbiviz' file into your report as guided by Microsoft.

  5. You can also download a sample report that includes the custom visual and showcases its capabilities.

For more details about the visual, formatting options and other information, please visit Advanced Visuals documentation.
